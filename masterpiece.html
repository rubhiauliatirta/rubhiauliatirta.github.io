Programs and Apps

TAMS (2019)

Software ini merupakan project software dengan Pertamina RTC yang digunakan untuk menghitung anomali thermal pada lapangan migas dengan menggunakan metoda pertabocsy. Fitur yang tersedia pada software ini antara lain terdiri dari interface input, interpretasi gambar mudlog, hingga pemetaan persebaran anomali themal dengan menggunakan metoda geostatistik.

C# Windows Presentation Fondation XAML Visual Studio

CatatHutang (2018)

Aplikasi berbasis android ini merupakan projek pribadi yang dibuat untuk mempelajari penggunaan Firebase Database. Fungsi dari aplikasi ini adalah untuk menyimpan catatan utang-piutang antar pengguna secara online dengan fitur notifikasi pengingat pribadi dan notifikasi untuk mengingatkan pengguna lainnya.

Java Firebase Database XML Android Studio

MoviePedia (2017)

Aplikasi berbasis android ini merupakan projek pribadi yang dibuat sebagai tugas dari kursus online "Menjadi Android Developer Expert" oleh dicoding. Aplikasi ini dapat melihat berbagai informasi film menggunakan API database dari themoviedb.org. Selain dapat melihat informasi film, pengguna juga dapat menyimpan film favorit, pengingat rilis film, dan juga menonton trailer.

Java XML Android Studio SQlite

DGS Intermediate Surface (2016)

Software ini berupa plugin untuk membuat peta intermediate surface dengan menggunakan data litologi pada sumur.Program akan mendata semua sumur yang terdapat pada projek petrel. Kemudian dengan batas horizon dan fasies litologi yang ingin diketahui ketebalannya, program akan menghitung ketebalan facies setiap titik. Fitur utama dari plugin ini adalah pengguna dapat membuat titik-titik ketebalan sendiri secara mudah agar hasil peta menjadi lebih "geologi".

Ocean Petrel C# Petrel Visual Studio 2010

TirStrat (2016)

Software ini merupakan produk dari tugas akhir yang saya kerjakan sekaligus pertama kalinya saya menyentuh pemrograman berbasis desktop dan OOP bersama dengan TSPCalculator. Software ini bertujuan untuk melakukan penggamabaran profil stratigrafi secara detail dari data yang diambil dari lapangan sesuai dengan format ITB. Gambar dapat diedit pada program ini sehingga dapat menampilkan informasi-informasi struktur sedimen serta suksesi vertikal dengan baik dan mudah.Hasil akhir dapat di export ke format pdf.

WinForm C# Visual Studio

TSPCalculator (2016)

Aplikasi ini merupakan produk tambahan yang dibuat sebagai produk dari skripsi yang saya susun. Manfaat dari software ini adalah untuk mengkuantifasi perhitungan porositas pada pada gambar sayatan batuan (thin-section) karena biasanya seorang geologis melakukan perkiraan dari penampakan yang ia lakukan. Hasil dari aplikasi ini memberikan angka porositas yang lebih presisi.

WinForm C# Visual Studio
